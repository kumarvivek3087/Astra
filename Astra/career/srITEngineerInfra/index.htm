About the Role

Join us as a Sr. IT Engineer (Infrastructure) and contribute to designing, maintaining, and securing the organization's IT ecosystem. You will work with servers, firewalls, networks, and cloud services while collaborating with cross-functional teams to ensure smooth business operations. This role is ideal for professionals with strong technical expertise in networking, security, and system administration.

Key Responsibilities

  • Windows Server Administration

    Hands-on experience with Windows Server 2012, 2019, 2022 including installation, configuration, and maintenance.

  • Firewall & Network Security

    Configure and manage Fortinet firewalls, VPNs, and perform firewall policy review and threat detection.

  • Routing & Switching

    Strong understanding of LAN, WAN, VLAN, routing, switching, and ability to work with Cisco L2/L3 devices.

  • Email & Cloud Administration

    Manage Exchange Server, Microsoft 365 Admin Center, and DLP administration.

  • IT Infrastructure & Hardware Management

    Manage Access Control, CCTV, Biometric systems, ILL, and complete hardware inventory.

  • Network Design & Implementation

    Understand network flow, create designs, and implement infrastructure solutions.

  • Active Directory & Azure AD

    Strong knowledge of Active Directory and Intra ID (formerly Azure AD).

Skills & Qualifications

  • Hands-on experience with Windows Server (2012/2019/2022)
  • Fortinet Firewall configuration and VPN setup
  • Knowledge of Cisco L2/L3 devices
  • Routing & Switching, LAN, WAN, VLAN
  • Exchange Server & Microsoft 365 Administration
  • Active Directory & Azure AD
  • Network design and implementation
  • Hardware infrastructure management

Preferred Skills

CCNA / CCNP certifications preferred
Experience with threat detection and firewall policy audits
